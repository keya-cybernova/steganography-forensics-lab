# 🔐 Steganography Forensics Lab
## 📌 Description
Digital Forensics project demonstrating steganography (hidden data in JPG) and recovery using FTK Imager.
## 🛠️ Tools Used
* SilentEye
* FTK Imager
* VirtualBox
## 🧪 Project Workflow
1. Created virtual disks (Data & Evidence)
2. Hid secret message inside JPG image
3. Stored file and deleted it
4. Recovered deleted file using FTK Imager
## 🎯 Result
Successfully recovered a deleted image containing hidden data.
## 📁 Folder Structure
/data
/evidence
## 🚀 Learning Outcome
* Steganography basics
* Digital forensics recovery
* File deletion vs actual data persistence
